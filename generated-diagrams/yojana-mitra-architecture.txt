DATA FLOW SUMMARY
=================

1. User speaks/types in Hindi/regional language
2. Transcribe converts voice to text (if needed)
3. API Gateway receives input
4. Bedrock Orchestrator analyzes query
5. Eligibility Agent queries Knowledge Base (RAG on scheme PDFs)
6. Explanation Agent simplifies response in user's language
7. Action Agent processes documents (Textract OCR) and generates forms
8. Polly converts text response to voice
9. Response delivered via WhatsApp/CSC/Web
10. Proactive Agent sends reminders via EventBridge + Lambda
